What is Static Site?

Static Site only includes content that can be loaded from a file(html, css, javascript, and images). It does not include any dynamic content that can be generated by server code.

Why do I prefer a static site?

  • Scalability
  • Reliability
  • Speed
  • Cheap to host
  • Version Control
  • More secure

When to use a static site?

static site approach is fantastic to

  • Landing Pages(or Homepages)
  • Informational Site
  • Promorional Site
  • Blogs
  • Portfolio(or Resume)
  • Small/Large Projects

Where to build a static site?

StaticGen

There's plenty of static site generator at StaticGen.com. you just have to choose what or which langauge is content written in. and most of them use Markdown.

How about Bootstrap?

Bootstrap
  • A Framework (A way to organise your code with a pre-built css styles and javascript).
  • Mobili first approach.
  • uses modern web technologies and is fully customisable.
  • Battle tested.

What is this Bootstrap Content Delivery Network (CDN)?

A place that hosts common library like bootstrap and JQuery.


- When someone visits a site that uses a cdn link, their browser will save its cache on the user's computer.
that means, his next visits to another site that used the same cdn is just like visiting internal link from the previous visit. It's very fast because it won't have to download the code again.
